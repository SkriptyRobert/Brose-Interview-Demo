# Brose_Job_Interview Demo
A simple web application builded in React that demonstrates containerized service deployment using Ansible, which sets up Podman, systemd, firewall, and SELinux, Haproxy. The application is hosted in a Podman container, running under a non-root user with automatic start-up on system boot. The setup includes a load balancer and multi-server configuration, with a PostgreSQL database for storing user data.
